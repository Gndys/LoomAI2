# 模特氛围图功能页线框与文案（V1）

> 用途：给产品/设计/前端直接落地页面结构与文案，不涉及代码实现。

## 1. 页面目标
- 让用户 30 秒理解“为什么要用模特氛围图”
- 让新用户 3 步完成首次出图
- 强化“模板化、批量化、可导出”的确定性

---

## 2. 页面线框（从上到下）

### A. 顶部价值条（Hero）
- 主标题：`3分钟生成可投放的模特氛围图`
- 副标题：`专为服装商家优化，白底图一键升级场景模特图`
- 主按钮：`立即上传商品图`
- 次按钮：`查看示例效果`
- 辅助文案：`新用户首周免费 20 张`

### B. 输入区
模块标题：`上传商品图`

文案与交互：
- 拖拽区主文案：`拖拽图片到这里，或点击上传`
- 支持说明：`支持 JPG/PNG/WEBP，单张不超过 15MB`
- 流转提示：`你也可以从白底结果一键带入图片`
- 操作按钮：`选择图片`、`清空队列`

列表字段：
- 文件名
- 分辨率
- 状态（待处理/处理中/完成/失败）
- 操作（重试/删除）

### C. 模板区（核心）
模块标题：`选择氛围模板`

默认提示：`已按转化场景排序，先选模板再生成`

模板卡片（V1）：
- `日常通勤感`：适合朋友圈日常上新
- `轻奢门店感`：适合私域成交和门店预热
- `街拍高级感`：适合小红书种草
- `电商影棚感`：适合详情页与主图补充

卡片信息：
- 模板名
- 一句适用说明
- 示例缩略图
- 标签：`新手推荐` / `高转化`

### D. 模特设定区
模块标题：`模特设置`

默认提示：`推荐设置已开启，通常无需修改`

字段：
- 性别：`女 / 男 / 中性`
- 年龄段：`18-24 / 25-30 / 30+`
- 体型：`纤细 / 标准 / 偏丰满`
- 姿态：`站姿 / 走动 / 半身`
- 表情：`自然微笑 / 冷感 / 不露脸`

按钮：
- 主按钮：`开始生成`
- 文本按钮：`恢复推荐设置`

### E. 结果区
模块标题：`生成结果`

工具栏：
- 筛选：`全部 / 通过 / 建议复检 / 失败`
- 排序：`最新处理 / 文件名`
- 视图：`网格 / 列表`

单卡片信息：
- 缩略图（结果）
- 角标：`通过` / `建议复检`
- 操作：`对比查看`、`再生成`、`下载`

对比弹窗：
- 标题：`原图 vs 模特氛围图`
- 滑块标签：`原图` `结果`
- 操作：`应用此版本`、`同模板再来一版`

### F. 质检区（可折叠）
模块标题：`质量检查`

检测项展示：
- 服装完整度：`完整 / 需复检`
- 人像自然度：`正常 / 异常`
- 背景干净度：`正常 / 有干扰文字`
- 商品细节可见度：`清晰 / 偏弱`

异常建议文案：
- `检测到手部细节异常，建议切换“电商影棚感”后重试。`

### G. 导出区
模块标题：`导出设置`

字段：
- 导出格式：`JPG / PNG`
- 图片质量：`标准 / 高清`
- 输出比例：`1:1 / 4:5 / 9:16`
- 命名规则：`原文件名_模板名_模特`（可编辑）

按钮：
- 主按钮：`批量导出（ZIP）`
- 次按钮：`仅导出已通过`

### H. 下一步引导区
模块标题：`继续完成宣发素材`

卡片：
- 标题：`生成电商海报`
- 描述：`把当前模特图一键套版成活动海报`
- 按钮：`去生成海报`

---

## 3. 状态页文案（必须具备）

### 3.1 空状态
- 标题：`还没有上传商品图`
- 描述：`上传 1 张商品图，几分钟得到可发的模特氛围图`
- 按钮：`上传第一张`

### 3.2 处理中
- 文案：`正在生成中，预计还需 {xx} 秒`
- 辅助：`你可以继续上传下一批图片`

### 3.3 失败状态
- 标题：`这张图生成失败`
- 原因示例：`检测到人像结构不稳定，已中断本次生成`
- 按钮：`重试一次`、`切换模板`

### 3.4 成功状态
- 标题：`已生成可用模特氛围图`
- 描述：`建议先预览服装细节与人像自然度，再批量导出`
- 按钮：`对比查看`、`批量导出`

---

## 4. 新手引导（首次进入）

引导步骤建议：
1. `上传商品图或导入白底图`
2. `选择一个模板直接生成`
3. `对比确认后批量导出`

首屏提示气泡：
- `先用“新手推荐”模板，成功率更高。`
- `这里可以快速筛出需要复检的图片。`
- `导出前可统一命名，方便团队协作。`

---

## 5. 转化文案（冷启动期）

- 信任文案：`已帮助 {X} 家服装商家完成上新素材生产`
- 价值文案：`不拍新片，也能稳定产出上新氛围图`
- 行动文案：`先出一批图，再决定是否付费`
- 收费文案：`首周免费 20 张，满意后再升级`

---

## 6. 数据埋点建议（供产品运营）

关键事件：
- 点击上传
- 上传成功
- 选择模板
- 点击开始生成
- 单图生成成功
- 批量导出
- 点击“去生成海报”
- 试用转付费

关键漏斗：
- 访问页面 -> 上传 -> 选模板 -> 生成 -> 导出 -> 次日复用 -> 付费
