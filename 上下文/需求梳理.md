- **原图已经包含了所有细节信息**
- **文字提示只用来"引导变化"**
- **描述越少，原图保留度越高**

```
【Nanobanana 提示词】

正向提示（Positive Prompt）：
flat lay product photography, white background, top view, centered, professional lighting, clean, minimal

负向提示（Negative Prompt）：
person, model, hands, face, body, UI elements, text, watermark, buttons, social media interface, shadows, wrinkles, accessories, background objects

【参数建议】

- 如果改变不够：提高到 0.5-0.7
```

### Step 3: 如果效果不满意

提供调整方案：

```
【问题诊断】

如果生成的图：
1. 和原图差太多 → 降低提示词强度到 0.2，或者简化正向提示
2. 还是有模特/背景 → 加强负向提示
3. 颜色偏了 → 在正向提示加：original color, accurate color
4. 细节丢失 → 降低强度 + 加：high detail, preserve details
5. 角度不对 → 强调：top-down view, 90 degree angle, bird's eye view
```
