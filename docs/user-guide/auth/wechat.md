# 微信扫码登录配置

本文档介绍如何配置微信扫码登录。

## 📑 目录

- [设置步骤](#设置步骤)
- [环境变量配置](#环境变量配置)
- [回调域名说明](#回调域名说明)
- [注意事项](#注意事项)

## 设置步骤

1. 访问 [微信开放平台](https://open.weixin.qq.com/)
2. 注册开发者账号并认证
3. 管理中心 => 网站应用 => 创建网站应用
4. 按邀请输入各种相关信息
5. 获取 AppID 和 AppSecret
6. 设置授权回调域: 网站应用 => 详情 => 开发配置 => 授权回调域

## 环境变量配置

在 `.env` 文件中添加：

```env
NEXT_PUBLIC_WECHAT_APP_ID="your_wechat_app_id"
WECHAT_APP_SECRET="your_wechat_app_secret"
```

## 回调域名说明

**本地开发环境**：
```
localhost:7001
```

**生产环境**：
```
yourdomain.com
```

**注意**：微信开放平台的回调域不需要写完整的路径后缀，只需要填写域名即可。

## 注意事项

- 微信扫码登录需要企业资质，个人无法申请
- 需要在微信开放平台完成企业认证
- 回调域名必须与实际访问域名一致
